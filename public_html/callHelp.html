Please see the Library of Congress classification system and the National Library of Medicine classification system for full explanations of their call-number systems and the links between call numbers and subjects.

You can also contact the subject specialist in the relevant discipline to get the call numbers in the subject you’re interested in.

There are four valid call number formats:

  • Call letter alone: select a call-letter from the drop-down and click add.
      This will match all call numbers that start with that exact call letter (e.g., "KF" matches KF1 – KF9999).
  • Call letter with wildcard: select a call-letter from the drop-down, click "Use a wildcard", and click add.
      This selects all call numbers that start with that letter (e.g., "K*" matches K1 – KZZ9999).
  • Call letter with start/base number: select a call-letter from the drop-down, click "Enter Range", enter a "Start Range/Base Number" and click add.
      This matches all call numbers that begin with that exact code/number combo (e.g., "K135").
  • Call letter with start and end ranges: select a call-letter from the drop-down, click "Enter Range", enter a "Start Range/Base Number" and "End Range" and click add.
      This matches all call numbers that fall within that range (e.g., "KF300 – KF352").
