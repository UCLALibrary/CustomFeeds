By default, all languages are included in the feed results.
If you want to limit the results to certain languages, select those languages from the list and click "Include Selected".
If you want to exclude certain languages from the results, select those languages from the list and click "Exclude Selected".
"Include Selected" and "Exclude Selected" are naturally exclusive:
If you limit the feed to, e.g., English and French, all other languages will be excluded; if you exclude Chinese and Polish, all other languages will be included.
