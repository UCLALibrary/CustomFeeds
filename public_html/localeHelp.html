By default, all libraries are included in the feed results.
If you want to limit the results to certain libraries, select those libraries from the list and click "Include Selected".
If you want to exclude certain libraries from the results, select those libraries from the list and click "Exclude Selected".
"Include Selected" and "Exclude Selected" are naturally exclusive:
If you limit the feed to , e.g., Arts and Music, all other libraries will be excluded; if you exclude Science and Engineering, all other libraries will be included.
